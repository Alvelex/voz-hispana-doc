# AnimationButtons

## Dónde está

```text
ReplicatedStorage.Shared.AnimationButtons
```

## Para qué sirve

Módulo común para animar botones de GUI con efectos visuales y sonido.

Debe usarse para mantener una experiencia consistente entre interfaces y evitar duplicar lógica de hover, click o sonidos en cada UI.

## Uso

Pendiente de documentar con el código real del módulo.

Cuando se revise este sistema, esta página debe incluir:

- cómo registrar un botón;
- qué sonidos/animaciones aplica;
- qué parámetros acepta;
- ejemplo mínimo de uso;
- errores comunes.

## Regla de equipo

Si una UI necesita botones animados, revisa primero este módulo antes de crear lógica propia.
