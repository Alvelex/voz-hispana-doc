# Voz Hispana Docs

Documentación técnica interna para desarrolladores del proyecto.

La documentación está pensada como una guía práctica: qué reglas seguir, dónde crear scripts y cómo usar los sistemas comunes sin necesitar entender toda la implementación interna.

## Publicación

Este repo usa Moonwave y GitHub Pages.

La web se publica automáticamente al hacer push a `main` mediante `.github/workflows/docs.yml`.
